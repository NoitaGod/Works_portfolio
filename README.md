1、扑翼机


作品亮点
上层机翼采用镂空后重新组合设计的方式凸显科技感
每层机翼都增加了外侧副翼，提高层次感
主机身设计骨节结构，逐节延伸，提高美观性
2、国风楼阁

作品亮点
所有横梁与立柱的连接处采用卯型和榫型设计
楼层结构和比例参考真实楼阁
提高了楼阁的整体高度，更显恢弘感
